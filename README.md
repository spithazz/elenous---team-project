A context-aware support application designed for individuals with early-stage dementia or Alzheimer’s disease. The system aims to support users in maintaining daily orientation and independence by assisting them based on their current context, such as location, time, social environment, and scheduled activities. The application provides a highly simplified and accessible patient interface, along with features for routine management, memory support, and caregiver integration for monitoring and notifications.

The system includes secure authentication with email and password, supporting two user roles: patient and caregiver. Caregivers are able to manage the patient’s daily activity program by creating and updating scheduled tasks. The application also provides medication reminders that notify patients at scheduled times and allow them to confirm medication intake. In addition, an emergency assistance feature enables patients to trigger immediate alerts to caregivers through a dedicated emergency button. A daily condition reporting function allows patients to submit their physical and emotional state once per day.

From an architectural perspective, the application follows the Boundary–Control–Entity (BCE) pattern, ensuring a clear separation of concerns between user interface components, control logic, and data entities. This supports modularity, maintainability, and scalability of the system.

The application prototype was designed in Figma

The project was developed as part of a Software Engineering course at the Aristotle University of Thessaloniki, Department of Electrical and Computer Engineering.
